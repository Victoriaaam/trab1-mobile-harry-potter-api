# Trab1 Mobile - Harry Potter API

Aplicativo Android desenvolvido para consultar a HP-API usando corrotinas e web services.

## Integrantes

- Nome 1
- Nome 2
- Nome 3

## Funcionalidades

- Buscar personagem por ID
- Buscar professor pelo nome
- Listar estudantes por casa
- Listar feiticos e ver detalhes

## API utilizada

HP-API

## Requisitos

- Android API 29
- Consultas em tempo real
- Repositorio no GitHub

## Estrutura inicial

- `MainActivity`: tela principal com botoes de navegacao.
- `BuscarPersonagemActivity`: tela da busca de personagem por ID.
- `BuscarProfessorActivity`: tela da busca de professor.
- `EstudantesCasaActivity`: tela da listagem de estudantes por casa.
- `FeiticosActivity`: tela da listagem de feiticos.
- `DetalhesFeiticoActivity`: tela de detalhes de um feitico.
